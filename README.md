# smart-food-label-analyzer
Interactive Python tool to analyze food nutritional values, calculate a health score, and provide real-time dietary recommendation
## Overview
This project analyzes food nutritional values and classifies them as Healthy, Moderate, or Unhealthy using a scoring system.

## Features
- Interactive user input
- Health scoring system
- Nutritional warnings
- Recommendation system

## Technologies
- Python

## How to Run
python main.py
